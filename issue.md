# Issue: Create Hero Section

## Objective
Implement a visually appealing and SEO-optimized hero section for the boarding house website.

## Requirements
- **Layout**: Split into two parts (side-by-side on desktop, stacked on mobile).
- **Left Side**:
    - SEO-optimized H1 heading.
    - Compelling description to encourage room bookings.
- **Right Side**:
    - Availability check form.
    - Fields: Start Date, End Date, People Capacity.
- **Background/Vibe**:
    - Incorporate visuals or themes suggesting safety, coziness, and a home-like atmosphere.
    - Highlight proximity to Pantai Gunung Payung or Pantai Pandawa.
- **Styling**: Modern, responsive, and using Tailwind CSS (consistent with the project followed style base on added color palette).
