# Issue: Create About Section

## Objective
Implement an informative and visually engaging "About" section for the Kuda Putih House website to highlight its facilities, services, and strategic location.

## Requirements
- **Content**:
    - **Introduction**: Describe Kuda Putih House as a premium boarding house located in Kuta Selatan, Badung, Bali.
    - **Facilities**: Showcase the available amenities:
        - 19 Boarding Rooms and 2 Suite Rooms.
        - Swimming Pool.
        - Common Kitchen (Dapur Umum) & Dining Area.
        - Mini Garden.
        - Grill BBQ Area.
    - **Extra Services**: Highlight value-added services:
        - Motor Bike and Car Rental.
        - Professional Tour Guide service for Bali excursions.
    - **Location**: Emphasize the strategic proximity to Pantai Gunung Payung and Pantai Pandawa.
- **Layout**:
    - Responsive design (mobile-first).
    - Use a grid or flex layout to organize text and facility icons/images.
    - Also add other features and benefits on Kuda Putih House like: free wifi, secure parking area, weekly cleaning,etc.
- **Styling**:
    - Modern, clean, and welcoming aesthetic.
    - Use Tailwind CSS consistent with the project's color palette and design system.
    - Incorporate icons to make the facilities list scanable.
    - Add 3 illustration images with grid layout, 1 image on the left with 2 rowspan, 2 images on the right.
- **SEO**:
    - Use semantic HTML tags (e.g., `<section>`, `<h2>`).
    - Include keywords like "boarding house Kuta Selatan", "Bali accommodation", "near Pantai Pandawa".
